## Features

- **Error Explanation**: Automatically detects errors and explains them in plain English
- **Diff Review**: Shows syntax-highlighted diffs of AI-generated fixes with explanations
- **Bug Dashboard**: Tracks bug patterns over time with category breakdowns

## Getting Started

1. Install the extension from VSIX
2. Set your Gemini API key: `VisualDebugger: Set Gemini API Key`
3. Open a JavaScript/TypeScript project and start coding
